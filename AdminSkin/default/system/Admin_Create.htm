缓存处理：
缓存处理：清空所有缓存　清空模版缓存
缓存设置：

1、缓存更新时间系数*： (推荐值:16-32 , -1:表示模版调试模式禁止缓存)

2、配置缓存模式*：

说明

缓存更新时间系数:
1:缓存时间系数越小,网页更新速度越快.数据库资源占用和CPU资源占用越大.反之系数越大占用数据库和CPU资源越少,但由于缓存时间使页面更新时间延长.
2:首页更新周期为:(缓存系数/8)上取整分钟 如:缓存系数为32,首页,列表页更新周期为32/8=4分钟

缓存模式说明:
1:自动模式:将自动使用内存+硬盘双重缓存模式,占用内存和空间都大,速度快.适合2G内存以上独立主机(VPS)使用.实际耗用资源大小和流量有关.
2:纯内存模式:只用内存用作缓存,速度快,适用于硬盘空间小环境.
3:纯硬盘模式(纯静态):适用于内存小但硬盘空间大环境,速度一般.)
